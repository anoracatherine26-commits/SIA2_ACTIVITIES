## Description of the Repository
This repository contains and organizes the compiled activities and outputs for the subject **System Integration and Architecture 2 (SIA 2)**. All activity files are properly organized according to their respective activities for easier navigation and evaluation.

## List and Explanation of the Activities Included



### Activity 2 – Titan Shop System
In this activity, I created **Titan Shop**, an automotive parts inventory system using Laravel. It includes product listings, search/filter functions, and a detailed product view.

### Activity 3 – Titan Shop CRUD System
This activity expands Titan Shop into a CRUD system for premium aftermarket automotive parts, featuring advanced search filters, real-time stock tracking, and secure checkout.

### Activity 4 – Titan Shop Form System
This activity builds a form-based e-commerce/inventory experience for premium automotive parts using Laravel, with a dark-themed UI for car enthusiasts.

### Activity 5 – Integration App System
This activity is an role-based integration app that streamlines user management and administrative oversight, directing users to appropriate dashboards.
